# 苏晓东前端React开发

## 项目技术栈

react + redux + vite + typescript + design-mobile

## 1. 项目本地运行

将项目 clone 至本地

```
cd react-test-01 (进入项目)
pnpm install (安装依赖包)
pnpm run dev (启动服务)
```
